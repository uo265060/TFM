TFM - Gestión de servicios TI de la Universidad de Oviedo con ServiceNow
ESTRUCTURA FICHERO:
./ -> Contiene el código implementado en la aplicación
./Acciones de IU -> Contiene un archivo XML (lpg_acciones_de_iu.xml) con el código de todas las acciones de IU creadas en el proyecto.
./Flujos de trabajo -> Contiene un archivo XML (lpg_flujos_de_trabajo.xml) con el código de todos los flujos de trabajo creados en el proyecto.
./Inclusiones de script -> Contiene un archivo XML (lpg_inclusiones_de_script.xml) con el código de todas las inclusiones de script creadas en el proyecto.
./Políticas de IU -> Contiene un archivo XML (lpg_politicas_de_iu.xml) con el código de todas las políticas de IU creadas en el proyecto.
./Pruebas -> Contiene un archivo XML (lpg_pruebas.xml) con el código de todas las pruebas unitarias y de regresión automáticas creadas en ServiceNow
./Reglas de asignación -> Contiene un archivo XML (lpg_reglas_de_asignacion.xml) con el código de todas las reglas de asignación creadas en el proyecto.
./Reglas de control de acceso (ACL) -> Contiene un archivo XML (lpg_reglas_de_control_de_acceso_acl.xml) con el código de todas las reglas de control de acceso (ACL) creadas y modificadas en el proyecto.
./Reglas de negocio -> Contiene un archivo XML (lpg_reglas_de_negocio.xml) con el código de todas las reglas de negocio creadas en el proyecto.
./Scripts de cliente -> Contiene un archivo XML (lpg_scripts_de_cliente.xml) con el código de todos los scripts de cliente creados en el proyecto.
./Scripts programados -> Contiene un archivo XML (lpg_scripts_programados.xml) con el código de todas los scripts programados creadas en el proyecto.
./Tablas creadas y modificadas -> Contiene un archivo XML (lpg_tablas_creadas_y_modificadas.xml) con el código de las tablas creadas y modificadas en el proyecto.
